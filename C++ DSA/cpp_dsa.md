# 🛣️ DSA Learning Roadmap

> ⚠️ **Important:** Don't jump directly into Graphs or Dynamic Programming.

Learn in the following sequence:

---

## 🟢 Phase 1 — Fundamentals

### 📌 Topics

- ⏱️ Time Complexity (Big O)
- 💾 Space Complexity
- 🔢 Arrays Basics
- 🔤 Strings Basics
- 🔁 Recursion Fundamentals

### 📝 Practice

- ✅ Linear Search
- ✅ Binary Search
- ✅ Basic Array Manipulation

---

## 🟢 Phase 2 — Traversal Patterns

> Traverse data structures efficiently.

### 📚 Topics

- Array Traversal
- String Traversal
- Matrix Traversal

### 📝 Problems

- ✅ Find Maximum Element
- ✅ Reverse Array
- ✅ Rotate Array

---

## 🟢 Phase 3 — Two Pointers

> ⭐ One of the highest-frequency interview patterns.

### 🕒 Use When

- 🟢 Sorted Arrays
- 🟢 Palindrome Problems
- 🟢 Pair Sum Problems

### 📝 Problems

- ✅ Two Sum II
- ✅ Remove Duplicates from Sorted Array
- ✅ Container With Most Water
- ✅ Valid Palindrome

---

## 🟢 Phase 4 — Sliding Window

> Extension of Two Pointers.

### 🕒 Use When

- 🟢 Subarray Problems
- 🟢 Substring Problems
- 🟢 Fixed Window
- 🟢 Variable Window

### 📝 Problems

- ✅ Maximum Sum Subarray
- ✅ Longest Substring Without Repeating Characters
- ✅ Minimum Window Substring

---

## 🟢 Phase 5 — Hashing

> Learn HashMap and HashSet thoroughly.

### 🕒 Use When

- 🟢 Frequency Counting
- 🟢 Fast Lookups
- 🟢 Duplicate Detection

### 📝 Problems

- ✅ Two Sum
- ✅ Group Anagrams
- ✅ Top K Frequent Elements

---

## 🟢 Phase 6 — Prefix Sum

### 📝 Problems

- ✅ Range Sum Query
- ✅ Subarray Sum Equals K

---

## 🟢 Phase 7 — Binary Search

> Binary Search is not just for searching.

### 📚 Topics

- Classic Binary Search
- Binary Search on Answer

### 📝 Problems

- ✅ Search Insert Position
- ✅ Koko Eating Bananas
- ✅ Capacity To Ship Packages Within D Days

---

## 🟢 Phase 8 — Linked Lists

### ⭐ Important Patterns

- Fast & Slow Pointer
- Reverse Linked List
- Cycle Detection

### 📝 Problems

- ✅ Reverse Linked List
- ✅ Middle of Linked List
- ✅ Linked List Cycle

---

## 🟢 Phase 9 — Stack

### 📚 Topics

- Stack Basics
- Monotonic Stack
- Next Greater Element

### 📝 Problems

- ✅ Valid Parentheses
- ✅ Daily Temperatures
- ✅ Largest Rectangle in Histogram

---

## 🟢 Phase 10 — Queue & Deque

### 📝 Problems

- ✅ Sliding Window Maximum
- ✅ Implement Queue Using Stacks
- ✅ BFS Basics

---

# 🌳 Phase 11 — Trees

## DFS Traversals

- 🌲 Preorder
- 🌲 Inorder
- 🌲 Postorder

## BFS Traversal

- 🌲 Level Order Traversal

### 📝 Problems

- ✅ Maximum Depth of Binary Tree
- ✅ Invert Binary Tree
- ✅ Binary Tree Level Order Traversal

---

# 🔄 Phase 12 — Backtracking

> Generate all possible solutions.

### 🕒 Use When

- 🟢 Generate all combinations
- 🟢 Constraint Satisfaction
- 🟢 Decision Trees

### 📝 Problems

- ✅ Subsets
- ✅ Permutations
- ✅ N Queens
- ✅ Sudoku Solver

---

# 🏔️ Phase 13 — Heap / Priority Queue

### 📝 Problems

- ✅ K Largest Elements
- ✅ Top K Frequent Elements
- ✅ Merge K Sorted Lists

---

# 🎯 Phase 14 — Greedy Algorithms

### 📝 Problems

- ✅ Jump Game
- ✅ Activity Selection
- ✅ Gas Station

---

# 🌐 Phase 15 — Graphs

### 📚 Topics

- ✅ BFS
- ✅ DFS
- ✅ Topological Sort
- ✅ Union Find (DSU)

### 📝 Problems

- ✅ Number of Islands
- ✅ Course Schedule
- ✅ Clone Graph

---

# 🔥 Phase 16 — Dynamic Programming

> Start DP only after becoming comfortable with recursion.

## Recommended DP Order

| Step | Topic |
|------|-------|
| 1️⃣ | Fibonacci |
| 2️⃣ | 0/1 Knapsack |
| 3️⃣ | Longest Common Subsequence (LCS) |
| 4️⃣ | Longest Increasing Subsequence (LIS) |
| 5️⃣ | DP on Trees |
